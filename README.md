# Ben-10---Top-10-Aliens
Ben 10 - Top 10 Aliens is an interactive slider-based website built using HTML, CSS, and JavaScript. It displays popular Ben 10 aliens in a modern card layout with smooth transitions and a futuristic design theme.  Each alien card contains an image, name, and a short description, making the website both visually appealing and informative.
Features

1) Smooth horizontal slider navigation

2) Left and right arrow controls

3) Clean card-based UI design

4) Stylish neon glow effects

5) Responsive layout for better viewing experience

This project is a creative frontend website showcasing modern UI design and JavaScript-based slider functionality.
